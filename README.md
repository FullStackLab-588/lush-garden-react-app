🌿 Plant Store – Dynamic React Web App

Plant Store is a modern, responsive web application built with React, TypeScript, and Tailwind CSS. The app is fully dynamic, fetching all content from APIs, making it scalable, maintainable, and production‑ready.

🚀 Live Demo

Live demo: https://lush-garden-react.netlify.app

📌 Features
Fully dynamic, API‑driven content
Fast, responsive, and mobile‑friendly
Clean and modern design with Tailwind CSS
Reusable components for scalability
Type‑safe development using TypeScript
🛠️ Tech Stack
React + TypeScript
Tailwind CSS for styling
React Router for navigation
React Hooks and Context API for state management
📁 Project Structure

The project is organized into folders for assets, components, pages, API services, custom hooks, TypeScript types, context, utilities, and global styles. This makes the codebase clean, maintainable, and easy to scale.

📡 API Integration

All content in the app is fetched dynamically from APIs. This includes plants, blogs, testimonials, and gallery items. Using API‑driven content ensures the app is fully dynamic and easy to maintain.

⚡ Setup & Run

Clone the repository, install dependencies, set up environment variables, and start the development server. The project runs locally and can be deployed to any hosting service.
